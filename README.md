# Titanic Data Analysis

## Project Overview
Analysis of Titanic passenger data to identify factors affecting survival rates. This project demonstrates basic data analysis and visualization skills using Python.

## Tools Used
- Python (Pandas, Matplotlib)

## Key Findings
- Women had a survival rate of 68% vs 32% for men.
- 100% of children in 1st and 2nd class survived, but only 40-50% of children in 3rd class survived.
- Passenger class had a significant impact on survival chances, even for children.

## Files
- titanic_analysis.py: Main analysis script.
- chart.py: Visualization of survival by gender.
- class_analysis.py: Survival by passenger class.
- age_analysis.py: Survival by age group.
- advanced_analysis.py: Detailed analysis by class, gender, and age.

## How to Run
1. Install Python 3.x
2. Install required libraries: pip install pandas matplotlib
3. Run any of the .py files
